ONE COMMUNITY MANAGER, TWO CANDIDATES

Bea Snchez (BS) y Alberto Mayol (AM) son dos candidatos del mismo conglomerado, el Frente Amplio (FA), compitiendo por ser el candidato presidencial de dicho grupo. Ambos hacen campaas en internet y tiene su propios medios, en el contexto de una mayor campaa que enfrentara a FA con otros conglomerados. Pero en esta primariaa quin apoya el community manager del FA?

Ok, tenemos las 745 publicaciones del community manager del FA en Facebook. Primero clasificaremos los post si se refieren a AM, a BS, a ambos o a ninguno.

posteo<-vector()

for (i in 1:length(FA$message)) {
  if ((grepl("mayol", FA$message[i], ignore.case = T) | grepl("alberto", FA$message[i], ignore.case = T)) && (grepl("bea", FA$message[i], ignore.case = T) | grepl("s.nchez", FA$message[i], ignore.case = T))) {
    posteo<-c(posteo,"Ambos")
  } else if (grepl("mayol", FA$message[i], ignore.case = T)|grepl("alberto", FA$message[i], ignore.case = T)) {
    posteo<-c(posteo,"Mayol")
  } else if (grepl("s.nchez", FA$message[i], ignore.case = T)|grepl("bea", FA$message[i], ignore.case = T)) {
    posteo<-c(posteo,"Bea")
  } else {
    posteo<-c(posteo,"Ninguno")
  }
}

FA$posteo<-posteo

Ahora calcularemos la cantidad de posteos por cantidato y la extensin promedio de cada texto como indicadores de preferencia.

Para contar palabras primero retiramos puntuacin y espacios dobles.

FA$message2<-str_replace_all(FA$message, pattern = "[[:punct:]]", " ")
FA$message2<-str_replace_all(FA$message2, "  ", " ")
FA$lista<-sapply(FA$message2, function(x) {strsplit(x = x,split = " ")})

FA$cuenta<-sapply(FA$lista, length)
extension<-aggregate(FA$cuenta, list(FA$posteo), mean)
cantidad<-aggregate(FA$cuenta, list(FA$posteo), length)
